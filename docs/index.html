はじめに

常微分方程式 (Ordinary Differential Equation, ODE) は, ただ一つの変数を持つ $1$ つの未知関数についての微分を含む方程式である. 例えば, $t$ を変数とする未知関数 $x(t)$ について, \[ \dfrac{dx}{dt} = f(t, x) \] といった形で表されるものは常微分方程式である, ただし, $f(x, t)$ は既知の関数であるとする. 以降, 記号の簡単のために \[ x' = \dfrac{dx}{dt} \] などと表記することがある.

一方で, (共通するただ一つの変数を持つ) $n$ 個の未知関数についての連立した常微分方程式を常微分方程式系 (System of Ordinary Differential Equations, System of ODEs), もしくは 連立常微分方程式 という. 例えば, $n$ 個の未知関数 \[ \mathbf{x}(t) = \begin{pmatrix} x_1(t) \cr x_2(t) \cr \vdots \cr x_n(t) \end{pmatrix} \] に対して, \[ \mathbf{x}' = \mathbf{f}(t, \mathbf{x}) \] といった形で表されるものは常微分方程式系である, ただし \[ \mathbf{x}' = \dfrac{d\mathbf{x}}{dt} = \begin{pmatrix} \dfrac{dx_1}{dt} \cr \dfrac{dx_2}{dt} \cr \vdots \cr \dfrac{dx_n}{dt} \end{pmatrix},\quad \mathbf{f}(t, \mathbf{x}) = \begin{pmatrix} f_1(t, x_1, x_2, \dots, x_n) \cr f_2(t, x_1, x_2, \dots, x_n) \cr \vdots \cr f_n(t, x_1, x_2, \dots, x_n) \end{pmatrix} \] であり, $f_1, f_2, \dots, f_n$ は既知の関数であるとする.

さて, \[ \mathbf{x}' = \mathbf{f}(t, \mathbf{x}) \] という形の常微分方程式系は, $\mathbf{f}$ が与えられていれば, 初期状態から将来の全ての状態が決定される. このような特徴を持つ系 (システム) は 力学系 (dynamical system) と呼ばれる. 特に, 上の式において右辺の関数 $\mathbf{f}$ が変数 $t$ に依存しない場合, つまり \[ \mathbf{x}' = \mathbf{f}(\mathbf{x}) \] と表すことができる場合, この力学系を 自励系 (autonomous system) と呼ぶ. 典型的には, 変数 $t$ は時刻であるとみなされ, $\mathbf{x}$ はシステムの状態を表す変数 (状態変数) であり, 相空間 (phase space) もしくは 状態空間 上の点と解釈される. また $\mathbf{x}'$ は $\mathbf{x}$ の速度を表していると解釈されるため, ある時刻 $t$ で位置 $\mathbf{x}$ にある点は, 速度 $\mathbf{f}(t, \mathbf{x})$ の流れに従って移動するとみなすことができる. 自励系においては, この"速度" が時刻に依存しないため取り扱いが比較的容易であり, $\mathbf{f}(\mathbf{x})$ の大域的構造を調べさえすればシステムの状態がどのように時間変化するかを決定することができる.

ただし, 力学系が仮に自励系であったとしても, 解析的に解くことができる力学系は一部であり, 一般にはシステムの状態の時間変化を知ることは不可能である. 解の挙動を調べるために, 解の周期軌道や平衡点の安定性, ポアンカレ写像による簡約化など様々な概念, テクニックが導入されたが, 複雑な問題に対する解析的な取り扱いは未だ困難である.

一方で, コンピュータの登場以降は数値解析によるアプローチが急激に普及した. 特に カオス と呼ばれる複雑な挙動を, 比較的容易に観察できるようになったことで, 科学や工学など様々な分野における複雑な現象が, 力学系や分岐現象, カオスという文脈で解釈されるようになった. 現在では, 様々な現象が微分方程式による数理モデルにより説明され, その性質を数学的に調べることで現象の理解や予想に役立てられているが, 力学系という枠組みもその一つとして非常に重要なアイデアである.

この資料では, 特に非線形力学系の解析を目指し, そのために必要な概念を導入する. はじめに線形力学系を取り上げ, 各概念について具体例や数値計算例を通じて理解を深めた後に, 非線形系の理論的な難しさにも言及したい. 本資料では線形代数や微積分, 初歩的な数値計算手法については既知のものとして詳細は解説しないため, 必要ならば確認, 復習しながら読み進めることを想定している.
